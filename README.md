## export-qiita

export qiita to local

### start

```
git clone git@github.com:yu-suke-dev/export-qiita.git

cd export-qiita

yarn install

< change user name to your own Qiita's user id , in index.ts >

yarn start
```

#### with docker 

```
$ docker build -t export-qiita .
$ docker run -it -v `pwd`/output:/tmp/output export-qiita
```
